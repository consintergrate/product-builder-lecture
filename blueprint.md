
# Lotto Number Generator

## Overview

This is a simple web application that generates and displays a set of random lottery numbers. It's built with HTML, CSS, and JavaScript.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the numbers in a clean and modern interface.
*   Responsive design for different screen sizes.

## Current Plan

*   **HTML (`index.html`):**
    *   Set up the basic structure with a title, a container for the numbers, and a button.
*   **CSS (`style.css`):**
    *   Style the layout, numbers, and button for a visually appealing look.
*   **JavaScript (`main.js`):**
    *   Implement the logic for generating and displaying the random numbers.
